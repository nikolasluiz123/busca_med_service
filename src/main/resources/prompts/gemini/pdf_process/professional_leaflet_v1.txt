IMPORTANTE REQUISITO DE FORMATAÇÃO:
O seu retorno DEVE ser obrigatoriamente um objeto JSON estritamente válido. Você deve garantir que a estrutura não quebre.
Para isso, certifique-se de realizar o escape correto de todas as aspas duplas internas que aparecerem dentro dos valores de texto utilizando a barra invertida (exemplo: \"). Substitua também todas as quebras de linha reais por caracteres de escape literais (\n) dentro das propriedades de texto. Não utilize blocos de código Markdown (```json) ou qualquer outro texto fora do objeto JSON.
